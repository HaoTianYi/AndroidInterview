#### 三元运算符和if语句第二种格式的关系

所有的三元运算符能够实现的，if语句的第二种格式都能实现。

反之不成立。

如果if语句第二种格式控制的语句体是输出语句，就不可以。

因为三元运算符是一个运算符，必须要有一个结果返回，不能是一个输出语句
